# Pyramid Pattern Programs in JavaScript

This repository contains multiple pyramid pattern implementations written in JavaScript.
The purpose of this repository is to demonstrate **progressive improvement in logic, structure, and code readability** while practicing core JavaScript fundamentals.

---

## üìö Implemented Patterns

The following pyramid patterns are included:

- **Pyramid Stars**  
  Basic pyramid pattern using loops.

- **Reverse Pyramid**  
  Inverted pyramid structure using nested loops.

- **Pyramid Using `repeat()` Method**  
  Improved readability and cleaner logic using JavaScript string methods.

- **Hollow Pyramid**  
  Pattern with conditional logic to create hollow structure.

- **Hollow Pyramid Using Functions**  
  Refactored version focusing on:
  - Code reusability
  - Separation of concerns
  - Better structure and maintainability
